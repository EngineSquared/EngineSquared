Detailed Description

A strongly-typed identifier wrapper based on entt::id_type.

This structure provides a type-safe wrapper around EnTT's id_type, offering implicit conversion to the underlying type for seamless integration with EnTT components. It inherits from BasicId to provide null checking and other common ID operations.

Note
Supports implicit construction from entt::id_type values
Supports implicit conversion back to entt::id_type for compatibility
Size is guaranteed to match entt::id_type through static assertion
See also
BasicId for base functionality
StringId for string-based identifiers
