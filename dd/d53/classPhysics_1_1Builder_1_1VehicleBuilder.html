template<size_t N>
class Physics::Builder::VehicleBuilder< N >

Builder for creating vehicles with N wheels.

Currently only supports N=4 (4-wheel vehicles). Attempting to use other values will result in a compile-time error.

Usage:

VehicleBuilder<4> builder;
builder.SetChassisMesh(chassisMesh)
.SetWheelMesh(WheelIndex::FrontLeft, wheelMesh)
.SetWheelSettings(WheelIndex::FrontLeft, frontWheelSettings)
.SetDrivetrain(DrivetrainType::RWD)
.Build(core);
