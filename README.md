# Engine² (or Engine Squared)

## How to run the project

### Requirements

- [Xmake](https://xmake.io/#/)
- [Visual Studio](https://xmake.io/#/)

### Running

1. Open VS solution using [Visual Studio](https://xmake.io/#/) (/vsxmake2022/EngineSquared.sln)
2. (Optional?) Install Xmake VS extension
3. Run the project
