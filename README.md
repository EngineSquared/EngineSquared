# Engine² (or Engine Squared)

## How to run the project

### Requirements

- [Xmake](https://xmake.io/#/)
- [Visual Studio](https://xmake.io/#/)

### Running

1. Execute `xmake project -k vsxmake -m "debug,release"` inside repository folder
2. Open previously created VS solution using [Visual Studio](https://xmake.io/#/) (/vsxmake2022/EngineSquared.sln)
3. (Optional?) Install Xmake VS extension
4. Run the project
